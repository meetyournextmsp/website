About

We list hustings for the Scottish Parliament elections in May 2021.

We only list non-partisan events - the Electoral Commission has guidance on this (their terminology is "non-selective").

We crowd source events from the public - please let us know of any events by filling in this brief form.

Open Data

All our event data is available for others to reuse. The data is managed in a GitHub repository

  • A list of all events
  • An SQLite database of all the data

Open Source

All our event data is available in a GitHub repository.

A Python component parses this data and produces a SQLite database.

The website is written in Python Flask.
